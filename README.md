# 🇯🇵 Japan Stock Market Analysis Agent

日本株の市場環境を毎日分析するための「市場分析エージェント」です。

為替（ドル円）、米金利、米国株、コモディティなどを統合し、
「今の日本株は何で動いているのか」をわかりやすくレポートとして出力します。

---

## 🚀 特徴

- 市場レジーム判定（円安主導 / 金利主導 / 半導体主導 など）
- 日本株を動かす主因の特定（ドル円・米金利・米株）
- セクター別分析（追い風 / 逆風 / 中立）
- 経済イベントの整理
- 行動スタンスの提示
- CLI / API 両対応

---

## 🧠 分析対象

- 日本株（日経平均・TOPIX）
- 米国株（S&P500 / NASDAQ / SOX）
- 為替（USD/JPY）
- 金利（米10年債など）
- コモディティ（原油・金）
- ボラティリティ（VIX）
