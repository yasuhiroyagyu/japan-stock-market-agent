日本株向けの「市場分析エージェント」をPythonで作成してください。

目的は、毎日の日本株市場について、
「今の地合いは何か」
「何が相場を動かしているか」
「どのセクターに追い風・逆風があるか」
「今日注目すべきイベントは何か」
を、簡潔で実用的な日本語レポートとして出力することです。

単なるニュース要約ではなく、為替・金利・米国株・マクロイベントを統合して、日本株の市場環境を解釈するエージェントにしてください。

【作りたいもの】
- Pythonで動くアプリ
- 毎回実行すると、日本株市場分析レポートを生成する
- 将来的にAPIやWeb UIへ拡張しやすい構成にする
- まずはMVPとしてCLI実行でよい
- 余裕があればFastAPIでAPI化してください

【対象市場】
- 日本株
- 米国株（日本株への影響要因として）
- 為替（USD/JPY）
- 米金利
- コモディティ
- ボラティリティ
- 日本株の主要セクター

【必須機能】
1. 市場データ取得
以下のデータを取得する関数を作ってください。
取得元は実装しやすいAPIまたはダミーデータでもよいですが、後でAPI差し替えしやすい形にしてください。

優先データ：
- 日経平均
- TOPIX
- 東証グロース指数
- USD/JPY
- S&P500
- NASDAQ
- SOX指数
- 米2年金利
- 米10年金利
- 日本10年金利
- WTI原油
- 金
- VIX

可能なら追加：
- 東証33業種の騰落率
- 売買代金
- 騰落レシオ
- 日銀関連イベント
- 米CPI/FOMC/雇用統計などのイベント情報

2. 市場レジーム判定
以下のような市場状態をルールベースで判定してください。
例：
- 円安主導の輸出株優位
- 金利上昇によるグロース逆風
- 半導体主導相場
- 商社・資源株優位
- リスクオン
- リスクオフ
- イベント待ち相場
- 方向感乏しい相場

判定は完全な精度でなくてよいので、まずはわかりやすいルールベースで実装してください。

3. 相場を動かしている主因の抽出
以下の要因のうち、その日に特に強く効いているものを3つ程度選んでください。
- ドル円
- 米10年金利
- 米株
- SOX
- 原油
- VIX
- 日銀政策
- 米経済イベント
- 半導体/AIテーマ
- 景気敏感/ディフェンシブ選好

4. セクター別影響分析
以下の形で出力してください。
- 追い風セクター
- 逆風セクター
- 中立セクター

ルール例：
- 円安 → 自動車・機械・輸出関連に追い風
- 米金利上昇 → 高PERグロース逆風、銀行追い風
- 原油上昇 → 商社・資源関連追い風、消費や空運逆風
- SOX上昇 → 半導体関連追い風
- VIX上昇 → 小型株やグロース逆風

5. 日本語レポート生成
最終的に以下のフォーマットで日本語レポートを出してください。

# 本日の日本株市場サマリー
- 市場レジーム:
- 総評:

# 重要変化
- ①
- ②
- ③

# 日本株を動かしている主因
- 要因1:
- 要因2:
- 要因3:

# セクター別インパクト
- 追い風:
- 逆風:
- 中立:

# 今日の注目イベント
- イベント名:
- 注目ポイント:
- 想定される影響:

# 行動スタンス
- スタンス1
- スタンス2
- スタンス3

さらに以下のスコアも0〜100で出してください。
- 日本株リスクオン度
- 円安追い風度
- 金利警戒度
- グロース逆風度
- 景気敏感優位度

【実装方針】
- まずはルールベースで実装
- API未接続でも動くよう、サンプルデータやダミーデータで動作確認可能にする
- 後で本物のAPIに差し替えやすい構造にする
- 関数やファイル分割をして、保守しやすくする
- コメントはわかりやすく書く
- 初心者でも読めるコードにする

【希望ディレクトリ構成】
以下のような構成で作成してください。

project/

  config.py
  requirements.txt
  services/
    market_data.py
    event_data.py
  logic/

    sector_analysis.py

  models/
    types.py
  sample_data/
    mock_market.json

【各ファイルの役割】
- main.py: 全体実行
- config.py: 設定値や定数
- services/market_data.py: 市場データ取得
- services/event_data.py: イベント取得
- logic/regime.py: 市場レジーム判定
- logic/sector_analysis.py: セクター影響分析
- logic/scoring.py: 各種スコア算出
- reports/generator.py: 日本語レポート生成
- models/types.py: データ構造定義
- sample_data/mock_market.json: API未接続時のサンプルデータ

【出力してほしいもの】
- 完成コード一式
- requirements.txt
- 実行方法
- どこをAPIに差し替えればよいかの説明
- 初心者向けの簡単な説明

【重要】
- 日本株においては「ドル円」「米10年金利」「米国株」「SOX」を特に重要視してください
- 単なる値の列挙ではなく、因果関係を説明してください
- 投資助言ではなく、市場理解の補助ツールとして設計してください
- 出力は日本語で簡潔に、朝3分で読めるレベルにしてください
